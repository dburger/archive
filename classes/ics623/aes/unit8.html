AES is a little complicated. However, the draft standard, which is in the duplicated material that I gave you, gives an example with a generous amount of debug printing, and if you use that effectively, you should be able to debug your program without great difficulty.

To make it a little simpler for you, I have made tables for the most common 8-bit operations. I actually wrote a program that made these tables, and I will show you the program and explain it in class. There are five tables. The first two are a logarithm and antilog tables. They are using a polynomial m(x) = x^8 + x^4 + x^3 + x + 1. It turns out that this is an irreducible polynomial, so working mod m(x) you get the field GF(256). However, this is not a primitive polynomial, i. e. its roots are not primitive elements of GF256). Calling a root of m(x) alpha, I found that beta = alpha+1 is a primitive element. Suppose e is any element of GF(256), and e = beta^i. Then L[e] = i and F[i] = e. Using this table you can multiply easily. if e1 = beta^i1 and e2 = beta^i2, then e1*e2 = beta^((i1+i2)mod255). (Note that for all elements except zero e^255=1.) Then you can multiply as follows: ans = F[(L[e1]+L[e2})%255); You can do various other things using those tables, too. You can calculate inverses, but for convenience, I made a table I[e] that gives the inverse of e. I also made a table SB[e] that does the S-Box function and SBI[e} that does its inverse. /* program for enciphering and deciphering AES. WP--7/01 */
#include <stdio.h>
int sb(int);
int M = 0x11b; /* the polynomial m(x) =  x^8 + x^4 + x^3 + x + 1 */
int L[256],    /* GF(256) logarithm      */
    F[256],    /* GF(256) antilogarithm  */
    I[256],    /* GF(256) inverse        */
    SB[256],   /* S-Box                  */
    SBI[256];  /* S-Box Inverse          */
void main()
{
    int i;
    int e = 0;
    int f = 1;
    /* If a is a root of M(x), a is not a primitive element of    */
    /* GF(256). However I found that 1+a is a root. I started     */
    /* with 1 and multiplied by 1+a at each step to get all of    */
    /* the field elements.                                        */
    for(i = 0; i<256; i++) {
        F[i] = f;
        L[f] = i;
        f = (f<<1) ^ f;
        if(f & 0x100) f ^= M;
    }
    F[255] = 1;
    L[0] = I[0] = -1;  /* invalid--shouldn't ever be used */
    for(i = 1; i<256; i++) {
        I[i] = F[255 - L[i]];
    }
    for(i = 0; i<256; i++) {
        SB[i] = sb(i);
        SBI[sb(i)] = i;
    }
    
    printf("/* GF(256) logarithm      */\n");
    printf("int L[256] = {\n");
    printf("-1 ");
