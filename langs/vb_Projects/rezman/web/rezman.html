Rezman 0.99

Rezman 0.99, the reservation management system for the short term touring industry.

The Program

Rezman 0.99 is a reservation management system designed for the short term touring industry. Originally designed for the helicopter tour industry of Kauai, it is also capable of managing the reservations for any business that schedules passengers on short term tours. Rezman 0.99 has an intuitive user interface and is easy to use.

The Download

A limited edition of Rezman 0.99 is available for download. While it contains all of the features that are in the full version, it was built with un-registered ActiveX controls that cause annoying dialog boxes to pop up while using the program. To download Rezman 0.99 click here. The download is a self-extracting executable file. To install Rezman 0.99 double click on rezdownload.exe. This will cause it to self-extract into three files, one of which will be called setup.exe. After the files have extracted, double click on setup.exe to begin the installation process. With the program installed, you should navigate the calendar to August 3rd, 1999. You will find some sample data on that date.
