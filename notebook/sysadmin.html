set up system on hailstorm:

1. sudo mkdir ordnance (/var/www)
2. sudo chown dburger:developers ordnance
3. sudo chmod 775 ordnance
--. capify .
4. cap deploy:setup
5. sudo chown -R dburger:developers *
6. cap -q deploy:check
6.5 mkdir -p shared/config
7. chown dburger:developers config
8. sudo chmod 755 config
9. create database
10. GRANT ALL ON database.* TO 'username'@'localhost' IDENTIFIED BY 'password'
11. put database.yml in place
